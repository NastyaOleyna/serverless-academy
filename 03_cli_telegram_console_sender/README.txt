Hello. This is a simple Telegram Console Sender.

In order to start the application you need to type "npm start" or "node index.js" in the console,
then go to the bot https://t.me/cli_tg_console_sender_bot 
and press the start button or enter "/start" in a chat with the bot. 

After that you can enter commands.

To find out what commands are available, type "npm start help" or "node index.js help".
